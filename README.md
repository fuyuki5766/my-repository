# my-repository
このファイルは、JR西日本などで採用されている電光表示板のようなものを再現できるジェネレーターである。
「index.php」では打ち込んだ文字が電光表示板本体にそのまま表示される他、「signbord.php」ではよりリアルな電光表示板のような構成となり、最大4列車までの列車案内が電光表示板本体に表示される。

## (使い方)
1.「index.php」について
ひらがなやカタカナはほぼすべての文字に対応できているが、漢字についてはデータに登録されていない文字も多く、打ち込んだ文字全てが電光表示板に反映されるとは限らない。

2.「signbord.php」について
「種別」「乗車位置」「発車時刻」「行先」「遅れ」の5つの選択項目がある。すべての項目においてプルダウン選択を採用しており、すべて選択必須となっている。

「種別」項目は「空欄」を選択することも可能。
(2026年6月2日現在で対応している種別)
普通、快速、新快速、特急、回送、通過

「行先」項目について、「種別」項目を「通過」「回送」としている場合、「行先」項目は空欄で固定され、選択することができない。「種別」項目を「通過」「回送」以外にしている場合は「行先」項目についても必須項目となる。
(2026年6月2日現在で対応している行先)
大阪、新大阪、高槻、京都、大津、長浜、西宮、神戸、西明石、姫路、新三田、谷川、福知山、宮津、和田山、天王寺、和泉府中、関西空港、海南、白浜、新宮、王寺、奈良、高田、木津、松井山手、岡山、宇野、福山、三原、広島、宮島口、下関、松江、浜田、高松、松山、宇和島、長崎、中津、大分、宮崎、福井、小松、金沢、高岡、七尾、長岡、新発田、村上、秋田、高山、名古屋、春日井、中津川、浜松、松本、長野、小田原、東京、上野、大宮、小山、宇都宮、水戸、いわき、八戸

「乗車位置」「発車時刻」の2つの項目について、「種別」項目を「通過」「回送」としている場合もしくは「行先」項目を「当駅止」(※今後実装予定)としている場合、「乗車位置」「発車時刻」の2つの項目は空欄で固定され、選択することができない。「種別」項目を「通過」「回送」以外にしている場合もしくは「行先」項目を「当駅止」以外にしている場合は「乗車位置」「発車時刻」項目についても必須項目となる。
「乗車位置」項目についても2つの選択項目があり、「乗車位置目印」と「乗車位置番号」に分かれている。「乗車位置目印」項目は「白△」「白〇」「黄↑」の3つを選択可能となっており、「乗車位置番号」については6両、7両、8両、10両、12両に対応している。
「発車時刻」項目についても2つの選択項目があり、「発車時」と「発車分」に分かれている。言わずもがな「発車時」は「0」～「23」、「発車分」は「00」～「59」に対応している。

「遅れ」項目については必須項目ではない。ただし、空欄にする場合は「なし」「あり」のチェックボックスのところで「なし」と選択する必要がある。
「あり」にチェックを入れた場合、遅れ時分を半角数字で入力する必要がある。

その他、一番下には「その他案内」項目があり、「index.php」のように打ち込んだ文字が電光表示板本体にそのまま表示されるが、全て電光表示板に反映されるとは限らない。
